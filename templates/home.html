Analisis Sentimen

Program ini melakukan analisis sentimen dan deteksi sarkasme terhadap data input. Program akan mendeteksi sarkasme yang terdapat dalam data, kemudian mengklasifikasikan data berdasarkan sentimen data, yaitu positif, negatif, atau netral. Data yang terdeteksi sebagai sarkasme akan diklasifikasikan ke sentimen negatif. Program ini menerima input berupa file csv
